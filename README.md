# **University Coursework: 1st Year, 2nd Semester**

This repository contains my academic projects and laboratory works for the second semester. The codebase is divided into three main categories: advanced data structures, an individual OOP project, and interactive graphics using SFML.

## **1. Algorithms and Data Structures (ACD_LABB_*.cpp)**
A collection of algorithms and complex data structure implementations written in C++. 
**Key topics covered and implemented from scratch:**
* Singly and Doubly Linked Lists
* Trees and Graphs
* Voronoi Diagrams
* String Search Algorithms (Knuth-Morris-Pratt, Boyer-Moore)
* Hash Tables

## **2. Individual Semester Project (ICS_ACD_FINALL.cpp)**
My final semester assignment
* **Core tech:** C++, Structs, Classes, and custom Singly Linked Lists.
* Focuses on memory management and structured data processing.

## **3. SFML Graphics & Game Dev Project (KP_PROG / arena / image)**
*Files:* `KP_PROG_*.cpp`, `arena_H.cpp`, `arena_H.h`, `image.cpp`, `image.h`
A course project demonstrating the practical application of C++ with the **SFML**.
* Demonstrates the separation of logic and headers (`.h` / `.cpp`) in a multimedia application.
